# 🎮 Ragnarok Online GeoGuessr

Mini-juego inspirado en **GeoGuessr**, pero ambientado en el mundo de **Ragnarok Online** 🧙‍♂️⚔️.  
El jugador debe adivinar en qué mapa se encuentra a partir de imágenes recortadas de escenarios del juego.  

Este proyecto nació como experimento con IA para generar parte del código, pero ha ido evolucionando como práctica personal para mejorar mis habilidades en **JavaScript, manejo del DOM y APIs**.

---

## 🚀 Características actuales
- Muestra imágenes aleatorias de diferentes mapas de Ragnarok Online.  
- El jugador debe seleccionar su respuesta entre varias opciones.  
- Retroalimentación instantánea (correcto/incorrecto).  
- Contador de puntuación para varias rondas.  

---

## 🛠️ Tecnologías utilizadas
- **JavaScript (Vanilla)**  
- **HTML + CSS**  
- **Imágenes de Ragnarok Online** (uso educativo/no comercial).
